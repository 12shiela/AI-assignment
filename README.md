# AI Algorithms Homework

This repository contains sample Python implementations of three important computational algorithms:

- **Genetic Algorithm (GA):** Evolutionary optimization inspired by natural selection.
- **Particle Swarm Optimization (PSO):** Swarm intelligence method based on bird flocking and fish schooling.
- **Markov Decision Process (MDP):** Framework for sequential decision-making under uncertainty.

Each algorithm includes **two sample codes** that demonstrate how they work in practice.
